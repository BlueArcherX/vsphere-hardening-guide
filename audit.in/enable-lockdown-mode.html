Audit Check enable-lockdown-mode

Lockdown mode is available on any ESXi host that is managed by a vCenter Server. Enabling lockdown mode disables all remote access to ESXi hosts, except for vCenter and vCLI management. Any subsequent local changes to the host must be made:

Lockdown mode can be enabled or disabled in two places:

• In the vSphere Client, when connected to the VMware vCenter Server managing the host
• In the DCUI of the host

Audit Check:

To check this manually, in the vSphere Client navigate to the configuration tab of the target host host. In the security profile setting, select Lockdown Mode, Edit, and verify that the checkbox for Lockdown Mode is set. 
In PowerCLI:
     $result =  $esxh  | Select Name,@{N="Lockdown";E={$_.Extensiondata.Config.adminDisabled}}
