Audit Check disable-mob

The Managed Object Browser (MOB) can be used to explore the object model used by vCenter to manage the vSphere environment. This can also be subverted to make configuration changes directly

Especially in a DMZ setting, this should be disabled

Audit Check:

psexec \\<vcenter ip> -u <userid> -p <password> cmd /c (cd "c:\programdata\vmware\vmware virtualcenter\") & `
      (powershell -Command "[xml]$vpxd = Get-Content vpxd.cfg" ; "write-Host $vpxd.config.enableDebugBrowse")

or
Browse to https://<vcenter ip>/mob

or in PowerShell:
    $uri = "https://" + $vc.name + "/mob"
    $result = invoke-webrequest $uri -credential $vicreds

    $compliance = $compliant

    if (($result.statuscode -eq 200) -or ($result.statucode -eq 401)) {
       $Compliance = $notcompliant
