Audit Check disable-unnecessary-functions

It is a general best practice on any host, physical or virtual, that if a service is not in use it should be disabled or removed. For instance, a web server typically does not need a DNS service or an SMTP mail service running on it – they would be good candidates for disabling or removal. Turn off any screen savers. If using a Linux, BSD, or Solaris guest operating system, do not run the X Window system unless it is necessary.
