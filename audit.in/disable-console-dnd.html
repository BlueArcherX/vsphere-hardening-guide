Audit Check disable-console-dnd Disable DND

This setting disables "drag and drop" using the shared folders between the VMware host and any VMs.

The affected VMX settings are: isolation.tools.dnd.disable=TRUE

Audit Check:


    vmrun -T esx -h https://<host ip>/sdk -u <userid> -p <password> readvariable <path to vmx file> isolation.tools.dnd.disable=TRUE
    (note that this can only check running VMs, use vmware-cmd for dormant hosts)

In PowerCLI:
    $vm | Get-AdvancedSetting -Name  "isolation.tools.dnd.disable"

A returned value of "TRUE" or blank indicates Compliance, "False" indicates non-compliance
