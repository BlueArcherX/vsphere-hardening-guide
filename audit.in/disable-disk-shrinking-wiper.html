Audit Check disable-disk-shrinking-wiper - Prevent Virtual Disk Wipe

The Disk Wiping operation can be executed by a user within a guest VM, by using the commands within the VMware Tools application. While this can be useful in clearing disk sectors, it is a CPU intensive operation that can affect not only the intended VM, but all VMs on the host platform. For this reason, disabling this feature is recommended. This is done by adding this line to the VMX file for the affected VM:
isolation.tools.diskWiper=TRUE

Audit Check: 


vmrun -T esx -h https://<host ip>/sdk -u <userid> -p <password> readvariable <path to vmx file> runtimeConfig isolation.tools.diskWiper
(note that this can only check running VMs, use vmware-cmd for dormant hosts)

In PowerCLI:
    $vm | Get-AdvancedSetting -Name  "isolation.tools.diskWiper.disable"

"TRUE" results are Compliant
"FALSE" or empty results are Non-compliant
