Audit Check disable-ssh

SSH is DISABLED by default. Enabling this access allows remote SSH access to the host, using root credentials.

Audit Check:

In PowerCLI;
     $esxh | Get-VMHostService | Where { $_.key -eq "TSM-SSH" } | Select VMHost, Key, Label, Policy, Running, Required

Manual Check

On the DCUI of a host, in the Troubleshooting Options menu, check the state (enable or disable - 
default disabled)


With the vSphere Client, on the Configuration tab for a host in the Security Profiles 
section, check the state - enable or disable (default disabled)

or:
nmap -p 22 --open
