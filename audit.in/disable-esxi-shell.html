Audit Check HCN06 - Disable Tech Support Mode

Tech Support mode is DISABLED by default. Enabling this mode allows Local and/or SSH access to the host, using root credentials.

Audit Check:

Manual Check

 On the DCUI of a host, in the Troubleshooting Options menu, check the state (enable or disable - default disabled) 

In PowerCLI:
    $esxh | Get-VMHostService | where  { $_.key -eq "TSM" } | Select VMHost, Key, Label, Policy, Running, Required

Manual Check:
With the vSphere Client, on the Configuration tab for a host in the Security Profiles section, check the state - enable or disable (default disabled)
